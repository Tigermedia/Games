# טופס הגדרת סוכן AI - AI Agent Configuration Form

אפליקציית ווב מודרנית ומקצועית בעברית (RTL) המאפשרת ללקוחות להגדיר את סוכן הצ'אטבוט האוטומטי שלהם.

## תכונות עיקריות

✨ **ממשק משתמש מודרני ומקצועי**
- תמיכה מלאה בעברית RTL
- עיצוב נקי ואינטואיטיבי
- צבעים מקצועיים (כחולים ואפורים)
- טיפוגרפיה נקייה עם פונטים שתומכים בעברית

📊 **מעקב התקדמות**
- אינדיקטור התקדמות ויזואלי
- ניווט קל בין סעיפים
- מעבר חלק בין דפים

💾 **שמירה אוטומטית**
- שמירה ב-localStorage כל 30 שניות
- אין חשש לאיבוד מידע
- אפשרות שמירה ידנית

📱 **רספונסיבי למובייל**
- עיצוב Mobile-First
- מתאים לכל גדלי מסך
- תפריט צד מתקפל במובייל

📤 **ייצוא נתונים**
- ייצוא לקובץ JSON
- שליחה באימייל
- העתקה ללוח
- מוכן להדפסה

## מבנה הטופס - 6 סעיפים ראשיים

### סעיף 1: שם הסוכן
- שדה טקסט יחיד לשם הסוכן
- דוגמאות למילוי
- שדה חובה

### סעיף 2: תיאור הסוכן
- שדה טקסט גדול (2-3 משפטים)
- מונה תווים
- שדה חובה עם ולידציה

### סעיף 3: פרסונה ותפקיד
- 3.1: תפקיד
- 3.2: אחריות עיקרית (5 שדות)
- 3.3: סגנון תגובה (טון, שפה, אימוג'י, אורך)
- 3.4: תכונות אישיות

### סעיף 4: מיומנויות
- 4.1: יכולות ליבה (צ'קבוקסים)
- 4.2: תרחישי עבודה (4 תרחישים עיקריים + נוספים)

### סעיף 5: מידע על מוצרים ושירותים
- סקירה כללית
- מפרטים טכניים
- מחירים
- אפשרויות הזמנה
- קישורים למקורות

### סעיף 6: אילוצים ומגבלות
- נושאים לדיון (ירוק)
- נושאים אסורים (אדום)
- מידע סודי (צהוב)
- הגבלות התנהגות
- כללי העברה לנציג אנושי

### סעיף 7: פרטי יצירת קשר
- שם מלא
- אימייל
- טלפון
- שם החברה

## התקנה והרצה

### דרישות מקדימות
- Node.js (גרסה 18 ומעלה)
- npm או yarn

### התקנת תלויות
```bash
npm install
```

### הרצת סביבת פיתוח
```bash
npm run dev
```

האפליקציה תרוץ על `http://localhost:5173`

### בניית גרסת ייצור
```bash
npm run build
```

הקבצים ייצרו בתיקייה `dist/`

### תצוגה מקדימה של גרסת ייצור
```bash
npm run preview
```

## טכנולוגיות

- **React 18** - ספריית UI
- **TypeScript** - שפת תכנות מוקלדת
- **Vite** - כלי בנייה מהיר
- **Tailwind CSS** - עיצוב וסטיילינג
- **React Hook Form** - ניהול טפסים
- **Zod** - ולידציה של נתונים
- **Lucide React** - אייקונים

## תכונות מתקדמות

### שמירה אוטומטית
הטופס נשמר אוטומטית כל 30 שניות ב-localStorage. המידע נשמר גם אם המשתמש סוגר את הדפדפן.

### ולידציה
כל שדה מוולדט באמצעות Zod לפני שליחת הטופס. הודעות שגיאה מוצגות בעברית.

### ייצוא נתונים
- **JSON**: הורד את כל הנתונים כקובץ JSON מובנה
- **אימייל**: פתח את לקוח האימייל עם התוכן מוכן
- **העתקה**: העתק את כל המידע ללוח

### הדפסה
הטופס מותאם להדפסה עם CSS ייעודי. כפתורים וניווט מוסתרים בהדפסה.
