# מדריך התחלה מהירה - טופס הגדרת סוכן AI

## 🚀 הרצה מהירה

האפליקציה כבר רצה וזמינה!

**כתובת:** http://localhost:5173

פשוט פתח את הדפדפן שלך וגלוש לכתובת המצוינת למעלה.

## 📋 מה כלול?

### ✅ כל התכונות מוכנות לשימוש:

1. **טופס מלא עם 7 סעיפים**
   - שם הסוכן
   - תיאור מפורט
   - הגדרת פרסונה ותפקיד
   - מיומנויות ותרחישי עבודה
   - מידע על מוצרים ושירותים
   - אילוצים ומגבלות
   - פרטי יצירת קשר

2. **ממשק משתמש מודרני**
   - תמיכה מלאה ב-RTL (עברית מימין לשמאל)
   - עיצוב נקי ומקצועי
   - צבעים כחולים ואפורים
   - אנימציות חלקות

3. **שמירה אוטומטית**
   - המערכת שומרת אוטומטית כל 30 שניות
   - שמירה ב-localStorage
   - לא תאבד נתונים גם אם תסגור את הדפדפן

4. **ניווט נוח**
   - אינדיקטור התקדמות
   - לחיצה על כל סעיף לדילוג ישיר
   - כפתורי "הבא" ו"הקודם"

5. **ולידציה מלאה**
   - בדיקת שדות חובה
   - הודעות שגיאה בעברית
   - מונה תווים לשדות ארוכים

6. **ייצוא נתונים**
   - הורדה כקובץ JSON
   - שליחה באימייל
   - העתקה ללוח

7. **רספונסיבי**
   - עובד מצוין במובייל
   - תפריט צד נפתח/נסגר
   - מותאם לכל גודל מסך

## 🎯 איך להשתמש?

1. **מלא את הטופס** - עבור בין הסעיפים ומלא את המידע
2. **שמור תוך כדי** - המערכת שומרת אוטומטית, או לחץ על כפתור "שמור"
3. **שלח את הטופס** - בסעיף האחרון לחץ "שלח טופס"
4. **ייצא את הנתונים** - בחר איך לקבל את המידע (JSON, אימייל, או העתקה)

## 📁 קבצים חשובים

- `src/App.tsx` - הקומפוננטה הראשית
- `src/components/` - כל הסעיפים של הטופס
- `src/types/form.ts` - הגדרות הטופס וולידציה
- `src/utils/storage.ts` - שמירה ב-localStorage
- `src/utils/export.ts` - ייצוא נתונים

## 💡 טיפים

1. **שמור לעתים קרובות** - גם אם יש שמירה אוטומטית, אפשר תמיד ללחוץ "שמור"
2. **השתמש בדוגמאות** - כל שדה מכיל placeholders עם דוגמאות לעזרה
3. **קרא את הטיפים** - תיבות המידע הכחולות/ירוקות מכילות עצות שימושיות
4. **ייצא בסיום** - אחרי שליחה, שמור עותק JSON לארכיון
